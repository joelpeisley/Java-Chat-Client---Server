How to run

To run the server app, go into src/server and run ChatServer after compilation:
ChatServer [port]
ChatServer 20000
This will start the server on port 20000 or whatever port is specified.

To run the client app, go into src/client and run ChatClient after compilation:
ChatClient [IPAddress] [Port]
ChatClient 127.0.0.1 20000
This will start the client and connect to a local server on port 20000.
